🔥 One-time Firebase Setup

Do this once — then editing is instant for all visitors forever.

1. Go to console.firebase.google.com → Create / open your project
2. Click ⚙️ Project Settings → "Your apps" → click </> Web icon → Register app
3. Copy the firebaseConfig object shown
4. Go to Firestore Database (left menu) → Create database → Test mode → Done
5. Open admin.html in VS Code or Notepad
6. Find YOUR_API_KEY and replace the entire config with yours
7. Do same in index.html
8. Upload both files to GitHub → your site updates automatically ✅
